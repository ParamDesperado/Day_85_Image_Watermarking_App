# 🖼️ Watermark Studio

A modern, browser-based watermark generator that lets you add professional text or logo watermarks to your images — completely free and offline.

🌐 **Live Demo:**  
[URL](https://day-85-image-watermarking-app.vercel.app/)

---

## ✨ Features

- 📝 Text watermarks with customizable fonts, colors, size, and opacity
- 🖼️ Image/logo watermark support
- 🎨 Smart contrast shadow for better visibility
- 📱 Responsive and modern UI
- 💾 Export watermarked images as PNG or JPG
- ⚡ Fast, client-side processing (no uploads required)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
- Vercel
